# 面接準備テンプレート - AI Product Development

## 🎯 面接準備の基本フロー

### 1. **プロジェクト初期化**
```
「面接準備」と入力
→ 面接日程、形式、面接官の情報を収集
→ プロジェクト構造を作成（Stock/programs/就職活動/projects/面接準備/）
→ Flow/YYYYMM/YYYY-MM-DD/就職活動_面接準備/ フォルダ作成
```

### 2. **プロジェクト憲章作成**
```
「プロジェクト憲章」と入力
→ 面接の基本情報（日程、形式、面接官、時間）
→ 志望企業・職種の詳細
→ プロジェクトの目標と成功基準
→ 制約条件とリスク管理
→ マイルストーン設定
```

### 3. **想定質問リスト作成**
```
「想定質問リスト」と入力
→ 職務経歴書の内容を参考に
→ 技術・アーキテクチャ関連の質問
→ AI・データ活用関連の質問
→ プロダクト開発・マネジメント関連の質問
→ 組織・チーム運営関連の質問
→ 志望企業特有の質問
→ 将来性・ビジョン関連の質問
```

### 4. **聞くべき質問リスト作成**
```
「聞くべき質問」と入力
→ 技術戦略・アーキテクチャ関連の質問
→ AI・データ活用関連の質問
→ 組織・チーム運営関連の質問
→ プロダクト開発関連の質問
→ 将来性・成長関連の質問
→ 具体的な業務内容についての質問
```

### 5. **企業研究**
```
「企業研究」と入力
→ 企業概要（基本情報、主要サービス、事業内容）
→ 組織構造（カンパニー制、技術組織、役員体制）
→ AI・技術戦略（AIトランスフォーメーション、技術的取り組み）
→ AI Product Developmentの役割
→ 技術的課題と機会
→ 面接での活用ポイント
```

### 6. **技術動向調査**
```
「技術動向調査」と入力
→ 最新AI技術動向（大規模言語モデル、マルチモーダルAI、エッジAI）
→ 志望企業のAI活用事例
→ AI Product Developmentの最新動向
→ 面接での活用ポイント
```

## 🎯 面接準備の実行手順

### 1. **初期設定**
```
「面接準備」と入力
→ 面接の基本情報を収集
→ プロジェクト構造を作成
→ プロジェクト憲章を作成
```

### 2. **準備フェーズ**
```
「想定質問リスト」と入力
→ 職務経歴書を参考に質問を作成
→ 回答ポイントを整理

「聞くべき質問」と入力
→ 技術・組織・ビジネス関連の質問を作成
→ 面接での活用ポイントを整理
```

### 3. **調査フェーズ**
```
「企業研究」と入力
→ 企業の基本情報を調査
→ 組織構造と技術戦略を理解
→ 面接での活用ポイントを整理

「技術動向調査」と入力
→ 最新のAI技術動向を調査
→ 企業のAI活用事例を調査
→ 面接での活用ポイントを整理
```

### 4. **練習フェーズ**
```
「回答練習」と入力
→ 想定質問への回答を準備
→ 技術的な深い理解を示す回答
→ ビジネス視点を含めた回答

「模擬面接練習」と入力
→ 90分間の集中力練習
→ 技術的な深い議論の練習
→ 質問する練習
```

### 5. **最終確認**
```
「面接チェックリスト」と入力
→ 当日の準備事項を確認
→ 技術的な質問への最終確認
→ 質問する内容の最終確認
```

## 📝 テンプレートの活用方法

### 1. **プロンプトとしての活用**
- 各セクションをコピーして使用
- 具体的な情報を入力してカスタマイズ
- 面接の種類に応じて調整

### 2. **チェックリストとしての活用**
- 各準備項目をチェック
- 完了した項目をマーク
- 不足している項目を確認

### 3. **参考資料としての活用**
- 面接当日の参考資料
- 技術的な質問への回答参考
- 質問する内容の参考
