# JIIMA認証申請に必要な事項まとめ

> 作成日: 2025-10-29
> プロジェクト: JIIMA認証申請

## 1. JIIMA認証とは

JIIMA（公益社団法人日本文書情報マネジメント協会）の認証は、電子帳簿保存法に基づくソフトウェアの法的要件適合性を認証する制度です。

## 2. 認証カテゴリ（5種類）

申請する製品・サービスが以下のどのカテゴリに該当するかを特定する必要があります：

1. **電帳法スキャナ保存ソフト法的要件認証**
2. **電子帳簿ソフト法的要件認証**
3. **電子書類ソフト法的要件認証**
4. **電子取引ソフト法的要件認証**
5. **電子契約ソフト法的要件認証**

## 3. 申請手順

### 3.1 事前準備
- [ ] 認証対象の特定（どのカテゴリに該当するか）
- [ ] 製品・サービスの機能要件の整理
- [ ] 法的要件との適合性の確認

### 3.2 申請書類の準備
- [ ] **申請書**：JIIMA公式サイトからダウンロード
- [ ] **機能チェックリスト**：該当ソフトウェアの機能が法的要件を満たしているかの確認
- [ ] **マニュアル**：ソフトウェアの機能や操作方法を説明する文書
- [ ] **その他必要書類**：JIIMAが指定する追加書類

### 3.3 申請・審査プロセス
1. **申請提出**：完成した申請書類をJIIMAに提出
2. **書類受付**：JIIMA事務局が申請書類を受け付け
3. **評価依頼**：事務局が評価機関に書類を送付
4. **評価実施**：評価機関が基準適合性を評価
5. **審査・判断**：審査委員会が認証の可否を判断
6. **結果通知**：申請者に認証結果を報告

## 4. 費用

### 4.1 審査手数料
- **新規審査手数料**：400,000円（税別）
  - JIIMA会員：320,000円（税別）
- **更新審査手数料**：250,000円（税別）
  - JIIMA会員：200,000円（税別）

### 4.2 その他費用
- **登録料**：100,000円（税別）
- **延長申請費用**：50,000円（税別）

### 4.3 割引制度
- 複数ソフトを同時申請する場合：総額から20%引き
- ※制度をまたぐ複数ソフトの同時申請は対象外

## 5. スケジュール

- **申請から認証取得まで**：通常3ヶ月程度
- **申請件数が多い場合**：さらに時間を要する可能性あり

## 6. 次のアクション

### 6.1 即座に必要な作業
- [ ] 申請対象の製品・サービスを特定
- [ ] 該当する認証カテゴリを決定
- [ ] JIIMA公式サイトで最新の申請書類をダウンロード
- [ ] 機能チェックリストの確認

### 6.2 準備作業
- [ ] 製品マニュアルの整備
- [ ] 法的要件との適合性確認
- [ ] 申請書類の記入・準備
- [ ] 予算の確保（審査手数料等）

### 6.3 申請後の作業
- [ ] 審査対応
- [ ] 追加資料の準備（必要に応じて）
- [ ] 認証取得後の運用準備

## 8. 注意事項

- 申請方法が変更されている場合があるため、最新の手順書を必ず確認
- 複数ソフトの同時申請を検討する場合は、割引制度を活用
- JIIMA会員になることで審査手数料の割引が可能
- 申請件数が多い時期は審査期間が延びる可能性がある
