# JIIMA認証申請 必要書類準備チェックリスト

> 作成日: 2025-10-29
> プロジェクト: JIIMA認証申請（電帳法スキャナ保存ソフト法的要件認証）

## 1. 必要書類一覧

### 1.1 必須書類
- [ ] **申請書**（オンラインフォーム入力）
- [ ] **機能チェックリスト（令和5年改正版）**
- [ ] **マニュアル**（ソフトウェアの機能と操作説明）
- [ ] **スキャン画像ファイル**（ISO 12653-3準拠テストチャート使用）

### 1.2 提出方法
- **オンライン申請**: JIIMA公式サイトの専用フォーム
- **書類提出**: 光ディスク郵送、メール添付、ファイル受け渡しシステム

## 2. 各書類の詳細準備手順

### 2.1 申請書（オンラインフォーム）

#### 準備事項
- [ ] **JIIMA公式サイトの確認**
  - URL: https://www.jiima.or.jp/certification/denchouhou/tetsuduki/
  - 最新の申請手順を確認
- [ ] **申請フォームの準備**
  - 会社情報の整理
  - ソフトウェア情報の整理
  - 連絡先情報の確認

#### 入力項目（予想）
- 申請者情報（会社名、住所、連絡先）
- ソフトウェア情報（名称、バージョン、機能概要）
- 申請カテゴリ（電帳法スキャナ保存ソフト法的要件認証）
- 提出書類の確認

### 2.2 機能チェックリスト（令和5年改正版）

#### 準備事項
- [ ] **チェックリストのダウンロード**
  - JIIMA公式サイトから最新版をダウンロード
  - 令和5年改正版であることを確認
- [ ] **ソフトウェア機能の確認**
  - 電子帳簿保存法の要件との適合性確認
  - 各機能の動作確認
  - テスト環境での検証

#### チェック項目（予想）
- **スキャン機能**
  - 解像度設定
  - ファイル形式対応
  - 圧縮設定
- **保存機能**
  - ファイル名規則
  - 保存先設定
  - バックアップ機能
- **検索機能**
  - キーワード検索
  - 日付範囲検索
  - ファイル形式検索
- **セキュリティ機能**
  - アクセス制御
  - ログ機能
  - 改ざん防止

### 2.3 マニュアル

#### 準備事項
- [ ] **マニュアルの作成・整備**
  - ソフトウェアの機能説明
  - 操作方法の詳細説明
  - 画面キャプチャの準備
- [ ] **提出形式の決定**
  - PDF形式
  - Word形式
  - ウェブページURL
  - 光ディスク

#### マニュアル内容
- **概要**
  - ソフトウェアの目的
  - 主要機能の説明
  - システム要件
- **インストール・設定**
  - インストール手順
  - 初期設定
  - 環境設定
- **操作方法**
  - スキャン操作
  - 保存操作
  - 検索操作
  - 管理機能
- **トラブルシューティング**
  - よくある問題
  - 解決方法
  - サポート情報

### 2.4 スキャン画像ファイル

#### 準備事項
- [ ] **ISO 12653-3準拠テストチャートの入手**
  - 購入先の確認
  - 入手方法の検討
  - コストの確認
- [ ] **スキャン環境の準備**
  - スキャナの準備
  - 設定の確認
  - テストスキャンの実施

#### スキャン要件
- **テストチャート**: ISO 12653-3準拠
- **解像度**: 指定された解像度でスキャン
- **ファイル形式**: 指定された形式で保存
- **スマートフォン対応ソフトの場合**:
  - Androidで撮影・取り込み
  - iOSで撮影・取り込み

## 4. 品質管理

### 4.1 チェックポイント
- **機能チェックリスト**: 全項目の確認・記入
- **マニュアル**: 内容の正確性・完全性
- **スキャン画像**: 指定要件の適合性
- **申請書**: 入力内容の正確性

### 4.2 レビュープロセス
- **内部レビュー**: 各書類の内容確認
- **技術レビュー**: 機能要件の適合性確認
- **最終レビュー**: 申請前の全書類確認

## 5. リスク管理

### 5.1 主要リスク
| リスク | 影響度 | 対策 |
|--------|--------|------|
| 機能チェックリストの不備 | 高 | 事前テスト、詳細確認 |
| マニュアルの不備 | 中 | 内部レビュー、外部確認 |
| スキャン画像の不適合 | 高 | 事前テスト、要件確認 |
| 申請フォームの入力ミス | 中 | 複数回確認、チェックリスト |

### 5.2 対策
- 各書類の事前テスト実施
- 複数人によるレビュー
- チェックリストの活用
- 余裕を持ったスケジュール
