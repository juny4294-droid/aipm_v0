# JIIMA認証申請プロジェクト計画 - スキャナ保存ソフト

> 作成日: 2025-10-29
> プロジェクト: JIIMA認証申請（電帳法スキャナ保存ソフト法的要件認証）

## 1. プロジェクト概要

### 1.1 認証カテゴリ
**電帳法スキャナ保存ソフト法的要件認証**

### 1.2 プロジェクト目的
電子帳簿保存法に基づくスキャナ保存ソフトの法的要件適合性認証を取得する

## 2. 申請手順

### 2.1 申請方法
- **オンライン申請**: JIIMA公式サイトの専用フォームから申請
- **紙ベース申請**: 不要（オンライン方式に変更済み）

### 2.2 審査プロセス
1. **申請書類提出** → JIIMA事務局
2. **評価機関審査** → 法的要件適合性評価
3. **審査委員会判断** → 電帳法スキャナ保存ソフト法的要件認証審査委員会
4. **認証結果通知** → 申請者への報告

## 3. 必要書類・準備事項

### 3.1 必須書類
- [ ] **申請書**（オンラインフォーム入力）
- [ ] **機能チェックリスト（令和5年改正版）**
- [ ] **マニュアル**（ソフトウェアの機能と操作説明）
- [ ] **スキャン画像ファイル**（ISO 12653-3準拠テストチャート使用）

### 3.2 マニュアル提出方法
- 電子ファイル
- 光ディスク
- URLで示されるウェブページ

### 3.3 スキャン画像ファイル要件
- **ISO 12653-3準拠 スキャナ評価用テストチャート**を使用
- **スマートフォン対応ソフトの場合**：
  - Androidで撮影・取り込みした画像ファイル
  - iOSで撮影・取り込みした画像ファイル

## 4. 費用

### 4.1 基本費用
- **新規審査手数料**: 400,000円（税別）
  - JIIMA会員価格: 320,000円（税別）
- **登録料**: 100,000円（税別）

### 4.2 追加費用
- **延長申請費用**: 50,000円（税別）

### 4.3 割引制度
- **複数ソフト同時申請**: 総額から20%割引
- ※制度をまたぐ複数ソフトの同時申請は対象外

## 5. スケジュール

### 5.1 審査期間
- **申請受理から認証まで**: 3～4ヶ月程度

### 5.2 認証有効期限
- **認証の有効期限**: 3年間

### 5.3 法改正対応
- 法改正があった場合、対応するための再審査が必要

## 6. 次のアクション

### 6.1 即座に必要な作業
- [ ] JIIMA公式サイトで最新の申請書類をダウンロード
- [ ] 機能チェックリスト（令和5年改正版）の確認
- [ ] ISO 12653-3準拠テストチャートの入手
- [ ] スキャン画像ファイルの準備

### 6.2 準備作業
- [ ] ソフトウェアマニュアルの整備
- [ ] 機能チェックリストの記入・確認
- [ ] オンライン申請フォームの準備
- [ ] 予算の確保（審査手数料等）

### 6.3 申請後の作業
- [ ] 評価機関からのフィードバック対応
- [ ] 追加資料の準備（必要に応じて）
- [ ] 認証取得後の運用準備

## 8. 注意事項

- 申請方法がオンライン方式に変更されているため、最新の手順を確認
- スマートフォン対応ソフトの場合は、Android・iOS両方の画像ファイルが必要
- 審査期間は3～4ヶ月程度と長いため、スケジュールに余裕を持って準備
- 法改正があった場合は再審査が必要となる可能性がある
