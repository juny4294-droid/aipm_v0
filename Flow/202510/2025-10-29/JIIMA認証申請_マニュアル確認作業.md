# JIIMA認証申請 マニュアル確認作業

> 作成日: 2025-10-29
> プロジェクト: JIIMA認証申請（電帳法スキャナ保存ソフト法的要件認証）
> ソフトウェア: ContractS CLM

## 1. マニュアル確認の目的

### 1.1 確認目的
- ContractS CLMソフトウェアの機能を詳細に把握
- 機能チェックリストの各項目に対する適合性を確認
- マニュアル記載箇所の特定
- 法的要件との適合性を検証

### 1.2 確認対象
- 操作説明書
- システム概要書
- 機能仕様書
- オンラインマニュアル・ヘルプ機能

## 2. 機能チェックリスト項目別マニュアル確認

### 2.1 訂正削除機能
- [ ] **A. 訂正削除の履歴を記録する方式**
  - スキャン文書と検索用データ登録機能
  - スキャン文書の訂正機能
  - スキャン文書削除機能
  - ファイル操作履歴機能
  - 訂正の履歴の検索、閲覧、出力
- [ ] **B. 訂正削除が物理的にできない方式**
  - スキャン文書と検索用データ登録機能
  - 法令に定められた保存期間中の訂正削除不可
  - 保存データがスキャニング直後のデータであることの証明

### 2.2 保存期間
- [ ] **保存期間管理機能**
  - 税法上の法定期間（7年間、または7年を越えて繰越欠損金の控除を行う場合はその控除期間中最長10年間）保存

### 2.3 相互関連性
- [ ] **属性管理機能**
  - 帳簿書類間の関連性を確保するための属性（伝票番号、取引案件番号、工事番号、URL等）
  - 書類単位で帳簿と書類間の関連性を確保するための属性設定
- [ ] **属性検索機能/リンク機能**
  - 帳簿書類間の関連性を確保するための属性による検索
  - 帳簿との関連性が無いという条件での検索

### 2.4 備付書類
- [ ] **操作説明書**
  - 書面もしくはデータやオンラインマニュアルやオンラインヘルプ機能での提供
- [ ] **システム概要**
  - システム全体の構成概要
  - スキャナで読み取った国税関係書類の保存機能の概要

### 2.5 可視性確保
- [ ] **ディスプレイ表示**
  - 14inch(35cm)以上、RGB256階調相当以上のディスプレイ対応
  - 帳簿書類に準じた規則性を有する形式での表示
  - 一覧性（原稿データを分割せずに表示）
  - 拡大・縮小表示
  - 表示品質（想定する最大サイズの原稿で1ページ全体を画面表示）
- [ ] **印刷機能**
  - 200dpi以上、RGB256階調相当以上のプリンタ対応
  - 帳簿書類に準じた規則性を有する形式での印刷
  - 一覧性（原稿サイズのまま分割せずに印刷）
  - 拡大・縮小印刷
  - 印刷品質（レシート大の原稿の拡大印刷、A3サイズの原稿の1枚印刷）

### 2.6 検索機能
- [ ] **基本検索機能**
  - 取引年月日その他の日付、取引金額及び取引先による検索
  - 一課税期間を通しての検索
  - 2つ以上の任意の条件をAND条件で検索
  - 値の範囲を条件にして検索
  - 値が入っていないことを明示的に検索
  - 一連番号等による検索
- [ ] **検索結果表示**
  - 検索条件にヒットしたもののみの一覧表示
  - 各書類の記録事項の確認
  - 名称での表記表示
- [ ] **検索結果印刷**
  - 検索結果の紙への印刷
  - 各書類の記録事項の確認
  - 名称での表記印刷

### 2.7 ダウンロード機能
- [ ] **データダウンロード**
  - CSV/TXT形式でのダウンロード
  - PDF形式でのダウンロード

### 2.8 装置等
- [ ] **解像度・階調要件**
  - 解像度200dpi以上・RGB256階調以上の画質要件
  - JIS X6933又はISO12653-3準拠スキャナ評価用テストチャートの4ポイント文字認識
  - 書類の解像度または画素数、階調及び書類の大きさに関する情報の保存

### 2.9 タイムスタンプ機能
- [ ] **タイムスタンプ付与**
  - 総務大臣が認定する時刻認証業務に係るタイムスタンプの付与
- [ ] **タイムスタンプ検証機能**
  - タイムスタンプ付与後の電子データおよびタイムスタンプの変更の有無の確認
  - タイムスタンプの信頼性の問題（有効期限切れ）の検知
  - タイムスタンプの時刻の確認
- [ ] **タイムスタンプ一括検証機能**
  - 指定された複数のデータおよびタイムスタンプの一括検証

### 2.10 タイムスタンプ代替機能
- [ ] **時刻情報取得**
  - 外部の第三者が提供する時刻情報の取得
  - パブリックNTPサーバからの日時取得
  - GNSS情報からの日時取得
  - 携帯電話の基地局からのNITZ情報取得
- [ ] **入力日時管理**
  - 利用者が電磁的記録の入力日時を変更できない機能
  - 電磁的記録の入力日時の参照・表示機能

### 2.11 システム移行機能
- [ ] **データ移行**
  - 移行元システム・サービスとしての移行対象データ、移行手順の記載
  - 移行先システム・サービスとしての移行対象データ、移行手順の記載
  - 移行元サービスとしての対応方針の記載
  - 移行先サービスとしての対応方針の記載

### 2.12 入力者情報
- [ ] **入力者記録**
  - 書類に係る記録事項の入力を行った者の氏名・部署名等の記録
  - 書類毎の入力者情報の確認

### 2.13 バックアップ機能
- [ ] **バックアップ機能**
  - データのバックアップ取得
  - 外部記録媒体へのバックアップデータ保存
  - バックアップデータの復元

### 2.14 機密性確保
- [ ] **アクセス管理**
  - 情報の利用範囲、更新履歴、機密度等に応じた管理区分設定
  - 情報にアクセスしようとする者の識別・認識
- [ ] **アクセス権限設定**
  - 利用者のアクセス権限設定
- [ ] **不正アクセス排除**
  - 不正なアクセスの排除
- [ ] **利用者ログ管理**
  - 利用者のアクセス記録のログ保存・追跡調査

### 4.1 適合項目の整理
- [ ] **完全適合項目**

- [ ] **部分適合項目**
  - 一部の要件を満たす項目
  - 追加機能が必要な項目
- [ ] **非適合項目**
  - 法的要件を満たさない項目
  - 機能が存在しない項目
