## ⚠️ 避けるべき逆質問

### ❌ NG質問例

1. **すぐに調べられる基本情報**
   - 「Splinkの事業内容を教えてください」
   - 「認知症とは何ですか？」
   - → 企業研究不足と判断される

2. **給与・待遇を最優先する印象**
   - 「残業代は全額支給されますか？」
   - 「昇給のタイミングを教えてください」
   - → 二次面接では避け、オファー後に確認

3. **ネガティブな印象を与える質問**
   - 「退職率はどのくらいですか？」
   - 「失敗したプロジェクトの事例を教えてください」
   - → 言い回しを工夫（「組織の課題」「改善中のポイント」など）

4. **Yes/Noで終わる質問**
   - 「リモートワークは可能ですか？」
   - → 「リモートワークとオフィス勤務のバランスについて教えてください」と深堀り型に

5. **面接官が答えられない範囲**
   - エンジニアに「CEOのビジョンは？」
   - CEOに「日々の細かい開発プロセスは？」

## 💡 逆質問のテクニック

### 1. **二段構えの質問**
最初に基本を聞き、その回答を受けて深堀り
```
例：
「Patient Journeyの統合で現在の課題を教えてください」
　↓（回答を聞いた後）
「その課題に対して、PMとしてどのようなアプローチが有効だと思いますか？」
```

### 2. **自分の経験を絡めた質問**
質問の前置きで自分の経験や考えを示す
```
例：
「私は以前、B2B2Cモデルでステークホルダー調整に苦労した経験があります。
Splinkでは購買決定者（医療機関）と利用者（患者）のニーズをどのようにバランスしていますか？」
```

### 3. **仮説を含めた質問**
自分なりの考えを示してから質問
```
例：
「認知症の早期診断ニーズが高まる中で、予防領域のプロダクトがより重要になると考えています。
Splinkとして、予防領域での事業拡大をどう見ていますか？」
```

### 4. **面接官の個人的な経験を聞く**
組織の公式見解だけでなく、リアルな声を引き出す
```
例：
「〇〇さんご自身が、Splinkで働く上で最もやりがいを感じる瞬間を教えてください」
```

### 5. **次のステップを意識した質問**
入社意欲を示しつつ具体的なイメージを持つ
```
例：
「もし私が入社した場合、最初の3ヶ月で特に注力すべきことは何でしょうか？」
```
