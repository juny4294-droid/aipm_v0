# オファー文章作成

## 概要
候補者のキャリアシートから魅力的なオファーレターを作成する

## 候補者ごとのフォルダ

オファー案・関連メモは **候補者別フォルダ** に保存します（このプロジェクトと並列の採用活動プロジェクト配下）。

- ルート: [../採用活動/candidates/](../採用活動/candidates/)
- フォルダ名は **`YYYY-MM-DD_氏名`**（作成日＋名前）。詳細は [candidates/README.md](../採用活動/candidates/README.md)
- 各候補者フォルダ内の `offer/` にレター原稿を置く運用を推奨
