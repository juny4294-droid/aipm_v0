# 採用活動

## 概要

採用活動の計画・実行・モニタリング・振り返りをまとめるプロジェクトです。求人設計、面接プロセス、オンボーディング連携など、採用業務全体を扱います。

## ドキュメント配置（PMBOKフェーズ）

| フォルダ | 用途 |
|---------|------|
| `documents/1_initiating` | プロジェクト憲章、ステークホルダー |
| `documents/2_discovery` | 前提・ペルソナ・課題定義など |
| `documents/2_research` | 市場・競合・候補者調査 |
| `documents/3_planning` | 計画、WBS、ロードマップ |
| `documents/4_executing` | 実行中の記録、議事録 |
| `documents/5_monitoring` | 進捗・KPI・変更管理 |
| `documents/6_closing` | 振り返り、ナレッジ移管 |

## 候補者別フォルダ

候補者ごとの資料（オファー案、面接メモ、添付など）は **`candidates/`** に、1人1フォルダで整理します。

- 運用ルール・命名（**作成日＋名前**）: [candidates/README.md](candidates/README.md)
- 新規追加時は [candidates/_template](candidates/_template) をコピー

## 添付・資料

**プロジェクト横断の**共通資料は `attachments/` に置いてください（候補者個人のものは `candidates/<名前>/` 側へ）。
